# DashFlow Launcher

DashFlow is a CarPlay-inspired Android launcher for in-vehicle head units. It replaces the stock Android home screen with a full-screen, landscape dashboard built around the things that matter while driving — a live map, speed, media playback, weather, and quick access to your apps.

Designed for low-RAM car head units, it runs lean without sacrificing a polished feel. Dashboards are fully customizable: arrange tiles, add pages, pin favorite apps to the dock, and theme everything to your liking. No internet account, no cloud sync, no bloat — just a clean interface between you and the road.
